# Engineering Standards

โฟลเดอร์นี้เก็บมาตรฐานและข้อกำหนดทางวิศวกรรม
ที่ใช้เป็นหลักในการออกแบบและติดตั้งระบบโซลาร์เซลล์ของ Lux Dee Solar

ขอบเขตเนื้อหา:
- มาตรฐานการเชื่อมต่อระบบไฟฟ้า (กฟภ. / กฟน.)
- มาตรฐานการติดตั้งทางไฟฟ้า (วสท. 2564)
- ข้อกำหนดด้านความปลอดภัย
- แนวทางที่ “ต้องทำ” และ “ไม่ควรทำ”

เอกสารในโฟลเดอร์นี้ใช้เป็น:
- Source of Truth ทางวิศวกรรม
- เอกสารอ้างอิงในการตรวจแบบและตรวจงาน
