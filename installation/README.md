# Installation Engineering

โฟลเดอร์นี้รวบรวมแนวทางการติดตั้งระบบโซลาร์เซลล์
ในมุมมองเชิงวิศวกรรม ไม่ใช่เพียงขั้นตอนช่าง

ขอบเขตเนื้อหา:
- Best practice งานติดตั้ง Solar Rooftop
- จุดเสี่ยงที่มักผิดพลาดหน้างาน
- แนวทางติดตั้งที่สูงกว่ามาตรฐานขั้นต่ำ
- บทเรียนจากงานจริงของ Lux Dee Solar

เป้าหมาย:
- ลดปัญหาหน้างาน
- งานติดตั้งปลอดภัย ตรวจรับผ่าน และดูแลต่อได้
