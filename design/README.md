# System Design & Engineering

โฟลเดอร์นี้ใช้สำหรับงานออกแบบระบบโซลาร์เซลล์
ตั้งแต่ระดับแนวคิดจนถึงแบบที่ติดตั้งได้จริง

ขอบเขตเนื้อหา:
- การคำนวณขนาดระบบ (Sizing)
- การจัด String และแรงดัน
- การเลือก Inverter / Battery / Protection
- Single Line Diagram (แนวคิดและตรรกะ)

เอกสารในโฟลเดอร์นี้ต้อง:
- สอดคล้องมาตรฐาน
- ใช้งานได้จริง
- อธิบายเหตุผลทางวิศวกรรมได้
