# Engineering Checklists

โฟลเดอร์นี้เก็บ Checklist สำหรับควบคุมคุณภาพงาน
ในแต่ละขั้นตอนของโครงการโซลาร์เซลล์

ขอบเขตเนื้อหา:
- Checklist ก่อนออกแบบ
- Checklist ก่อนติดตั้ง
- Checklist ก่อนจ่ายไฟ / Commissioning
- Checklist ก่อนส่งมอบงาน

Checklist ทุกชุดมีเป้าหมายเพื่อ:
- ลดความผิดพลาด
- ทำให้งานเป็นระบบ
- ตรวจย้อนหลังได้
